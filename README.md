# 🏛️ Museum Webpage with Chatbot & Ticket Booking

An interactive **museum website** featuring a smart chatbot that assists users with ticket booking, pricing details, and a simulated payment system. This project combines **frontend UI design** with **basic conversational logic** to enhance user experience.

---

## 🚀 Features

* 💬 **Interactive Chatbot**

  * Handles greetings and basic queries
  * Provides ticket prices and opening hours
  * Guides users through ticket booking

* 🎟️ **Ticket Booking System**

  * Select visit date
  * Choose number of tickets
  * Availability check (max limit per day)

* 💳 **Payment Simulation**

  * Card number validation
  * Expiry date and CVV validation
  * Booking confirmation after successful payment

* 🖼️ **Responsive Museum Webpage**

  * Clean UI with gallery section
  * Mobile-friendly chatbot interface
  * Smooth animations and transitions

## 💡 How It Works

1. Click the **chat icon** at the bottom-right
2. Ask questions like:

   * “ticket price”
   * “opening hours”
   * “book tickets”
3. Follow chatbot steps:

   * Enter date → Select tickets → Confirm booking
4. Complete **payment simulation**
5. Get booking confirmation 🎉

---

## 🔒 Validation Logic

* Card number: 13–19 digits
* Expiry date: Must be valid and not expired
* CVV: 3–4 digits
* Ticket limit: Max 5 per booking
* Daily capacity: Max 10 tickets
